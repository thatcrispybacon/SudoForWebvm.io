# Contribute
If you have a feuture that isn't available in [webvm.io](https://webvm.io) but is available in real debian or ubuntu feel free to **FORK** and submit a pull request.  
If you know of a feuture and you don't know a away of enabling it / accessing it on [webvm.io](https://webvm.io) post an issue and it will be investiagted on how this could be worked around.  

> [!IMPORTANT]
> If the methord you use may raise a vulnerability **especially in networking** please the dangers clear.
